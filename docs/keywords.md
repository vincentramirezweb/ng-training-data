# Keywords (clusters, intención y URLs)

## Cómo leer los archivos de keywords
- `data/keyword_map_consolidated.csv`: todo lo relevante consolidado.
- `data/keyword_map_business_owner.csv`: **mentorías para dueños** (recomendado para priorizar).
- `data/keyword_map_cleaning_courses.csv`: keywords de cursos/capacitación de limpieza.

Columnas:
- `avg_monthly_searches`: volumen estimado (Keyword Planner)
- `intent`: Informativa / Comercial / Transaccional
- `funnel_stage`: TOFU/MOFU/BOFU
- `target_url`: URL recomendada para atacar esa keyword
- `cluster`: temática
