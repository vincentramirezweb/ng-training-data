# KPIs y tracking (GA4 + GSC)

## KPIs principales
- Leads: envíos de formulario (conversion)
- Compras: purchase (WooCommerce)
- Comunidad: clics a Skool (event outbound_link)

## KPIs SEO (GSC)
- Impresiones / clics por URL:
  - /mentorias/
  - productos clave (Jobber, Premium)
- CTR por query BOFU (mentoría, jobber en español, marketing limpieza)
- Posición media por query

## Eventos recomendados GA4
- form_submit (o generate_lead)
- begin_checkout
- purchase
- outbound_click_skool (click URL contains skool.com/ng-training-6385)
