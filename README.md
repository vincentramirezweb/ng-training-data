## Qué incluye este repo
- `data/`  
  - `gsc_queries_domain.csv`: consultas (GSC) del dominio completo (incluye marca).  
  - `gsc_queries_nonbrand.csv`: consultas NO marca (filtradas).  
  - `gsc_pages_domain.csv`: desempeño por páginas (GSC).  
  - `keyword_map_consolidated.csv`: keywords + volumen (Keyword Planner / exports) + intención + URL objetivo.  
  - `keyword_map_business_owner.csv`: subset enfocado a mentorías para dueños.  
  - `keyword_map_cleaning_courses.csv`: subset enfocado a cursos/capacitación de limpieza.

- `docs/`
  - `strategy.md`: enfoque, oferta, funnel y prioridades.
  - `pages.md`: páginas clave y objetivo SEO por URL.
  - `keywords.md`: clusters y keywords por intención.
  - `conversions.md`: metas de conversión y eventos recomendados (GA4).
  - `schema.md`: reglas de schema (Yoast + SASWP) y qué aplicar por tipo de página.

---

## Nota importante sobre “volumen”
- Google Search Console **NO** entrega volumen de búsqueda (global).  
- Los archivos `Keyword Stats ...` (Keyword Planner) aportan **Avg. monthly searches** (estimado).  
- En `keyword_map_*` el volumen puede venir vacío si el export no lo trajo para esa keyword.
